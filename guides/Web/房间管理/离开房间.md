本文介绍通过 NERoom SDK 实现离开房间功能的方法。

## 配置步骤
1. 根据 roomUuid 获取 NERoomContext。
2. 调用`leaveRoom`接口离开房间。

  ```
  fun leaveRoom()
  ```
3. 成员离开房间成功后，会触发 `NERoomListener` 协议中的 `onMemberLeaveRoom` 回调方法，通知房间内所有成员。


## 示例代码

实现离开房间功能的示例代码如下：

```
/**
 * @param roomUuid 房间ID
 */
const NERoomContext = NERoomService.getRoomContext(roomUuid)

NERoomContext.leaveRoom().then(() => {
   console.log('leaveRoom success')
}).catch(err => {
   console.error(err, 'leaveRoom fail')
})
```
