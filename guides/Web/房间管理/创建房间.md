房间是 NERoom 产品的基石，搭建应用场景时，您首先需要创建房间。

## 注意事项

- 创建房间后，不会自动加入房间，您需要调用接口加入房间。
- PK 直播房间的保留时长最多为 24 小时，超过 24 小时，NERoom Server会自动删除该房间。
- 其他业务场景中，房间的保留时长最多为 16 小时，超过 16 小时，NERoom Server会自动删除该房间。
- 如果您需要延长默认的房间保留时长，请联系网易云信商务经理。

## 配置步骤

调用 [`createRoom`](https://doc.yunxin.163.com/neroom/references/web/typedoc/Latest/zh/html/interfaces/index.NERoomService.html#createroom) 接口创建房间。

```TypeScript
createRoom: (
    params: NECreateRoomParams,
    options: NECreateRoomOptions,
) => Promise<NEResult>
```

- `NECreateRoomParams` 相关参数说明如下表所示。

    名称 | 描述 |
    ---- | ---- | 
    roomUuid | 房间 ID，房间 ID 需要保证全局唯一。<br>最大长度为 64 个字符。 |
    roomName | 房间名。<br>最大长度为 64 个字符。 |
    templateId | 模板 ID。类型：number。模板 ID 的值如下：<ul><li>线上会议：40<li>互动直播：3<li>在线K歌：400<li>语聊房：569<li>自定义模板：模板 ID获取方法请参见 [如何获取模板 ID](https://doc.yunxin.163.com/neroom/docs/zM3MTQ0Nzk?platform=web#如何获取模板-id)。 |
    password |（可选）设置当前房间的密码。如果设置为空，表示用户进入房间时无需密码。|
    initialProperties |（可选）设置房间属性。|
    seatInitParams |（可选）设置麦位的初始化配置。包括如下参数：<ul><li>`seatCount`：麦位数量。默认为 8，最大为 20。<li>`seatRequestApprovalMode`：麦位申请审批模式。<ul><li>1：观众申请上麦时，需要管理员同意后才能上麦。<li>0(默认)：观众申请上麦时，管理员自动同意上麦。</ul><li>`seatInvitationConfirmMode`：麦位邀请确认模式。<ul><li>1：管理员邀请观众上麦时，需要观众同意后才会上麦。<li>0(默认)：管理员邀请观众上麦时，观众自动同意上麦。

- `NECreateRoomOptions` 相关参数说明如下表所示。

    名称 | 描述 | 
    ---- | ---- | 
    enableRtc| 设置该房间是否包含音视频通话的功能。<ul><li>true：包含音视频通话功能。</li><li>false：不包含音视频通话功能。</li></ul> | 
    enableChatroom | 设置该房间是否开启即时消息的功能。<ul><li>true：包含即时消息功能。</li><li>false：不包含即时消息功能。</li></ul> | 
    enableWhiteboard | 设置该房间是否包含互动白板的功能。<ul><li>true：包含互动白板功能。</li><li>false：不包含互动白板功能。</li></ul> |
    enableRecord | 设置该房间是否开启录制的功能。<ul><li>true：开启录制功能。</li><li>false：关闭功能。</li></ul> |
    enableLive | 设置该房间是否开启直播的功能。<ul><li>true：开启直播功能。</li><li>false：关闭功能。</li></ul> | 
    enableWaitingRoom | 设置该房间是否开启等候室的功能。<ul><li>true：开启等候室功能。</li><li>false：关闭功能。</li></ul> | 

## 示例代码

```TypeScript
/**
 * 创建房间
 * @param params.templateId 模板ID
 * @param params.roomName 房间名称
 * @param params.roomUuid 房间ID
 * @param options.enableRtc 是否需要配置RTC房间
 * @param options.enableChatroom 是否需要配置聊天室房间
 * @param options.enableWhiteboard 是否需要配置白板房间
 */

const params = {
    templateId: 'your templateId',
    roomName: 'your roomName',
    roomUuid: 'your roomUuid'
}
const options = {
    enableRtc: true,
    enableChatroom: true,
    enableWhiteboard: true,
    enableRecord: true,
    enableLive: true
}
NERoomService.createRoom(params, options).then(res => {
    console.log(res, 'createRoom success')
}).catch(err => {
    console.error(err, 'createRoom fail')
})
```
