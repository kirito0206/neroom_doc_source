## <span id="如何获取模板ID">如何获取模板 ID</span>
每个 NERoom 应用会绑定一个场景模板，模板 ID 就是您的 NERoom 应用绑定的场景模板的 ID。创建房间时您需要填入相应的模板 ID。

您可以在网易云信控制台获取对应的模板 ID。具体步骤如下：

1. 登录 <a href="https://app.yunxin.163.com/index#/">网易云信控制台</a>。
2. 在首页选择对应的应用。
3. 在左侧导航栏选择 **产品功能** > **房间组件模板**，在 **组件模板管理** 区域，单击对应模板的 **角色管理**。

   <img alt="角色管理.png" src="https://yx-web-nosdn.netease.im/common/39e59d0200b65c0d2d45c9a837f1976e/角色管理.png" style="width:60%;border: 0px solid #BFBFBF;border-radius:0px;">

4. 查看模板 ID 的值。

   <img alt="模板 ID.png" src="https://yx-web-nosdn.netease.im/common/59dfe5a54981c2ed0e6cb9665be3b9e2/模板ID.png" style="width:60%;border: 0px solid #BFBFBF;border-radius:0px;">

## <span id="如何获取角色ID">如何获取角色 ID</span>

每个 NERoom 应用会绑定一个场景模板，场景模板中可以设置角色。角色 ID 就是您的 NERoom 应用绑定的模板中的角色的 ID。

您可以在网易云信控制台获取创建房间时所需的角色 ID。具体步骤如下：

1. 登录 <a href="https://app.yunxin.163.com/index#/">网易云信控制台</a>。
2. 在首页选择对应的应用。
3. 在左侧导航栏选择 **产品功能** > **房间组件模板**，在 **组件模板管理** 区域，选择某个模板，单击 **角色管理**。

    <img alt="角色管理.png" src="https://yx-web-nosdn.netease.im/common/39e59d0200b65c0d2d45c9a837f1976e/角色管理.png" style="width:60%;border: 0px solid #BFBFBF;border-radius:0px;">

4. 查看各个角色 ID 的值。

   <img alt="角色 ID.png" src="https://yx-web-nosdn.netease.im/common/67382d6d1894a492a61b4e2ee14bffdf/角色ID.png" style="width:60%;border: 0px solid #BFBFBF;border-radius:0px;">

## <span id="如何修改角色的权限">如何修改角色的权限</span>

角色权限是在您的 NERoom 应用绑定的场景模板中定义的。
- 如果您的 NERoom 应用绑定的是系统自带的场景模板，例如互动直播模板，则无法修改角色的权限。
- 如果您的 NERoom 应用绑定的是自定义场景模板，您可以修改角色的权限。具体步骤如下：
    1. 登录 <a href="https://app.yunxin.163.com/index#/">网易云信控制台</a>。
    2. 在首页选择对应的应用。
    3. 在左侧导航栏选择 **产品功能** > **房间组件模板**，在 **组件模板管理** 区域，单击 **自定义方案** 中的 **角色管理**。

        <img alt="角色管理.png" src="https://yx-web-nosdn.netease.im/common/39e59d0200b65c0d2d45c9a837f1976e/角色管理.png" style="width:60%;border: 0px solid #BFBFBF;border-radius:0px;">

    4. 在 **角色管理** 区域，单击对应的角色，修改角色的权限。单击 **保存**。

        <img alt="修改角色.png" src="https://yx-web-nosdn.netease.im/common/d1a15e606aa7cdade595f84b08be18fe/修改角色.png" style="width:60%;border: 0px solid #BFBFBF;border-radius:0px;">

## <span id="修改角色的权限后，会立即生效吗">修改角色的权限后，会立即生效吗</span>

在 <a href="https://app.yunxin.163.com/index#/">网易云信控制台</a> 修改角色权限后，对已经创建的房间不生效，只对新创建的房间生效。

## <span id="noCompatibleVersionsForPod">编译 iOS 时，提示 “CocoaPods could not find compatible versions for pod ***” 相关错误</span>

<img alt="image.png" src="https://yx-web-nosdn.netease.im/common/54647ce041eda0d0488efdb5f5aa7a9e/image.png" style="width:60%;border: 1px solid #BFBFBF;border-radius:0px;">

这种错误一般是和其他组件一起使用时出现的（例如和 IM UIKit 组件同时使用了），可以在工程的 `ios` 路径下的 Podfile，添加如下配置：

```
Pod.const_set(:SPECIAL_VERSION, true)
Pod.const_set(:FCS_SPECIAL_VERSION, true)
pod 'NIMSDK_LITE/FCS', '9.20.13'    # 这里的配置仅是示例，可以参考报错进行修改为对应的版本
pod 'NECoreIMKit/FCS_Special', '9.6.10'
```
