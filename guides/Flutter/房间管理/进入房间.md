本文介绍通过 NERoom SDK 实现进入房间的方法。创建房间后，主播或观众进入房间参与互动。

## 配置步骤

1. 调用 `joinRoom` 接口加入房间。

```dart
Future<NEResult<NERoomContext>> joinRoom(NEJoinRoomParams params, NEJoinRoomOptions options);
```

`NEJoinRoomParams` 相关参数说明如下表所示。

名称 | 描述 |
---- | ---- |
roomUuid | 房间 ID |
userName | 用户名 |
role | 角色 ID，请在网易云信控制台上查看对应 NERoom 应用的角色 ID，具体请参见 [如何获取角色 ID](https://doc.yunxin.163.com/neroom/concept/DQzNTEwMDE?platform=client#获取角色-id)。|
password|（可选）当前房间的密码。|
initialProperties|（可选）设置成员属性。|

`NEJoinRoomOptions` 相关参数说明如下表所示。

名称 | 描述 |
---- | ---- |
noVideo | 配置入会时是否关闭本端视频，默认为 true，即关闭视频，但在房间中可重新打开 |
noAudio | 配置入会时是否关闭本端音频，默认为 true，即关闭音频，但在房间中可重新打开 |
extras | 额外选项 |

2. 通过 `NERoomService` 的 `getRoomContext` 方法获取房间上下文 `NERoomContext`。

3. 通过 `NERoomContext` 的 `addEventCallback` 方法，注册房间所需的监听事件，具体请参见 [监听房间事件](https://doc.yunxin.163.com/neroom/guide/zMyNTgyMjM?platform=flutter。

## 示例代码

实现进入房间功能的示例代码如下：

```dart
NEJoinRoomParams params = NEJoinRoomParams(
    role: role, roomUuid: roomUuid, userName: userName);
NEJoinRoomOptions options = NEJoinRoomOptions();
roomService.joinRoom(params, options).then((result) {
  if (result.isSuccess()) {
    debugPrint('joinRoom success');
  } else {
    debugPrint('joinRoom error code:${result.code},msg:${result.msg}');
  }
});
```
