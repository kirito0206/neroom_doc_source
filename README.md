# NERoom SDK 文档

欢迎使用 NERoom SDK 文档。此仓库包含将 NERoom SDK 集成到您的应用程序中的全面指南和参考。

## 文档结构

### 指南 (`/guides`)

所有支持平台的完整集成指南、教程和最佳实践：

- **Android** - 移动端 Android 集成
- **iOS** - 移动端 iOS 集成
- **Web** - Web 浏览器集成
- **Windows** - Windows 桌面端集成
- **macOS** - macOS 桌面端集成
- **Flutter** - 跨平台移动应用

每个平台指南包括：
- **快速开始** - 下载 SDK、运行示例、基本集成
- **房间管理** - 创建、加入房间和邀请
- **成员管理** - 房间内成员控制和权限管理
- **音视频通话** - 实时音频视频通信功能
- **即时消息** - 文本消息和其他消息类型
- **互动白板** - 白板协作功能
- **互动直播** - 直播和互动功能

### API 参考 (`/api-reference`)

*详细的 API 参考* - 访问：[API 参考](./api-reference/README.md)

## 开始使用

1. 从 `/guides` 目录中选择您的目标平台
2. 按照快速入门指南下载 SDK 并运行示例代码
3. 根据集成指南将 SDK 集成到您的项目中
4. 根据您的用例探索高级功能

## 关键特性

### 核心能力

- **多平台支持** - 支持 6 个平台的原生 SDK，包括移动、桌面和 Web
- **高质量音视频** - 专业级实时音视频通话
- **灵活架构** - 支持各种场景：
  - 一对一音视频通话
  - 多人小组房间
  - 预定房间和即时房间
  - 房间控制和成员管理

### 安全特性

- **房间安全** - 房间密码、访客验证
- **媒体加密** - 端到端加密传输
- **房间控制** - 音视频状态控制、房间锁定

### 业务组件

- **即时消息** - 文本消息、表情、文件传输
- **互动白板** - 共享白板协作功能
- **互动直播** - 低延迟直播互动
- **成员管理** - 成员角色与权限控制

### 高级功能

- **音视频增强** - 回声消除、降噪、美颜功能
- **麦位管理** - 麦位申请与管理机制
- **房间扩展** - 自定义房间功能扩展
- **IM 集成** - 重用现有 IM 账户系统
- **私有云** - 支持私有部署
