Class NEConnectedDevice

  • All Implemented Interfaces:
    
public final class NEConnectedDevice

    表示已连接到房间的设备信息。

    在多设备场景下，同一用户可能同时使用多个设备（手机、电脑、平板等）加入房间， 此类封装了每个已连接设备的基本标识信息。
