    NESeatRequestApprovalMode麦位申请审批模式。

    • OFF 审批模式关闭，即成员提交申请后不需要管理员同意，可直接上麦。

    • ON 审批模式打开，即成员提交申请后需要管理员同意才能上麦。

    NESeatInvitationConfirmMode麦位邀请确认模式。

    • OFF 确认模式关闭，即管理员邀请其他成员上麦，不需要被邀请的成员确认，可直接上麦。

    • ON 确认模式打开，即管理员邀请其他成员上麦，需要被邀请的成员确认，接受后方可上麦。
