int x
final

用户图像的横轴坐标值

用于设置用户图像的横轴坐标值。 通过 x 和 y 指定画布坐标中的一个点，该点将作为用户图像的左上角。 取值范围为 0~1920，若设置为奇数值，会自动向下取偶。

用户图像范围如果超出超出画布，调用方法时会报错。
