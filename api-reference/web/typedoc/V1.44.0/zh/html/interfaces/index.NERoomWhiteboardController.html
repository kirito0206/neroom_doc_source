        配置项

        • Optional content?: "clip" | "fitToDoc" | "fitToContent" | "region"

          clip导出当前页显示的内容。导出图片分辨率和容器大小一样

          • clip导出当前页显示的内容。导出图片分辨率和容器大小一样
          • fitToDoc导出图片和页面中课件的宽高一样
          • fitToContent导出图片和页面中内容的最大宽高一样。如果内容较小，会设置最小宽高为容器宽高
          • region导出选定的区域，并导出为指定的图片大小
        • Optional customBg?: boolean

          导出图片时，是否包含自定义背景。默认为false

        • Optional omitSelection?: boolean

          是否绘制选择框。默认为false，即不忽略（绘制）

        • Optional quality?: number

          图片精度。有效范围为(0, 1]。只有当type为'jpeg'时有效

        • Optional regionParam?: {
              cameraBound: {
                  centerX: number;
                  centerY: number;
                  height: number;
                  width: number;
              };
              size: {
                  height: number;
                  width: number;
              };
          }

          如果 content 为 region，则 regionParam 为必填项

          • cameraBound: {
                centerX: number;
                centerY: number;
                height: number;
                width: number;
            }
            • centerX: number
            • centerY: number

          • size: {

                width: number;
            }
            • height: number

        • Optional type?: "jpeg" | "png"

          导出图片的类型。"jpeg" | "png"
